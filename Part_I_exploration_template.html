What is the structure of your dataset?

The dataset consists of 16 columns (Features), where each row represents one trip using a Ford GoBike system bike during February 2019. There are data types: numeric like duration_sec, text like start_station_name, time like end_time, and category like user_type.

What is/are the main feature(s) of interest in your dataset?

I'm interested in understanding how long trips typically last, and whether trips differ between user types (subscribers vs. customers). Therefore, the main feature of interest is duration_sec, which represents the duration of each bike trip in seconds.

What features in the dataset do you think will help support your investigation into your feature(s) of interest?

Several features can help support this analysis:

start_time and end_time: to explore temporal patterns (e.g., peak hours, weekday vs weekend usage).

user_type: to compare trip duration between Subscribers and Customers.

import matplotlib.pyplot as plt
import seaborn as sns

def plot_histogram(data, column, max_value=None, bins=30, color='skyblue', xlabel=None, title=None):
    
    plt.figure(figsize=(10, 6))
    
    if max_value is not None:
        filtered_data = data[data[column] <= max_value]
    else:
        filtered_data = data

    sns.histplot(data=filtered_data, x=column, bins=bins, color=color, edgecolor='black')
    
    if title is None:
        title = f'Histogram of {column}'
    plt.title(title, fontsize=14)
    
    if xlabel is None:
        xlabel = column
    plt.xlabel(xlabel, fontsize=12)
    
    plt.ylabel('Count', fontsize=12)
    plt.grid(True)
    plt.tight_layout()
    plt.show()

import seaborn as sns
import matplotlib.pyplot as plt
import pandas as pd

# Clean and prepare data
data['member_age'] = 2025 - data['member_birth_year']
ordered_days = ['Monday', 'Tuesday', 'Wednesday', 'Thursday', 'Friday', 'Saturday', 'Sunday']
data['day_of_week'] = pd.Categorical(data['day_of_week'], categories=ordered_days, ordered=True)

# Filter for logical values
subset = data[
    (data['duration_min'] <= 60) &
    (data['member_gender'].isin(['Male', 'Female'])) &
    (data['user_type'].isin(['Customer', 'Subscriber']))
]

# Create FacetGrid
g = sns.FacetGrid(
    subset,
    col='day_of_week',
    col_wrap=4,
    height=4,
    aspect=1,
    sharey=True,
    sharex=True
)

# Map boxplots to each facet
g.map_dataframe(
    sns.boxplot,
    x='user_type',
    y='duration_min',
    hue='member_gender',
    palette='Set2'
)

# Add titles and labels
g.set_titles('{col_name}')
g.set_axis_labels('User Type', 'Trip Duration (minutes)')
g.add_legend(title='Gender')
g.fig.suptitle('How Trip Duration Varies by User Type and Gender Across Days', y=1.05)
plt.tight_layout()
plt.show()

We note through the analysis that customers take longer trips than male subscribers. We also find that female customers take the longest trips of all, especially on weekends (Saturday and Sunday). This indicates that they use the service for travel and entertainment. Meanwhile, weekday subscribers, especially males, take shorter trips. This indicates that they use the service frequently, such as commuting to work.
