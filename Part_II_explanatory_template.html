Investigation Overview

This analysis aims to understand bike-sharing system usage patterns by analyzing trip duration by user type, gender, and day of the week. The analysis aims to uncover user behaviors and discover relationships between variables to better understand how the service is used, which can help improve marketing planning and service development.

Dataset Overview and Executive Summary

The Ford GoBike dataset contains information on individual bike rides made in a bike-sharing system in the San Francisco Bay Area. The dataset includes various attributes such as trip duration, station locations, user demographics, and time-related details. Below are the key columns from the dataset:

  • duration_sec: Total duration of the trip in seconds.
  • start_time: Timestamp for when the ride started.
  • end_time: Timestamp for when the ride ended.
  • start_station_id: Identifier for the starting station.
  • start_station_name: Name of the starting station.
  • start_station_latitude: Latitude of the starting station.
  • start_station_longitude: Longitude of the starting station.
  • end_station_id: Identifier for the ending station.
  • end_station_name: Name of the ending station.
  • end_station_latitude: Latitude of the ending station.
  • end_station_longitude: Longitude of the ending station.
  • bike_id: Unique identifier for each bike.
  • user_type: Categorizes users as either a 'Subscriber' or 'Customer'.
  • member_birth_year: Birth year of the rider.
  • member_gender: Gender of the rider (Male, Female, or Other).
  • bike_share_for_all_trip: Indicates whether the bike was shared for all trips.
  • day_of_week: Day of the week when the trip was taken.
  • duration_min: Trip duration converted into minutes for easier analysis.
  • member_age: Calculated age of the rider based on their birth year.
  • distance_km: Estimated distance between start and end stations, calculated using their geographic coordinates. The dataset contains 20 columns and includes 183,215 individual bike rides. Each ride has details such as trip duration, start and end locations, user type, and demographics.

import matplotlib.pyplot as plt
import seaborn as sns

def plot_boxplot(data, x_col, y_col, max_y=None, palette='Set2', title=None, xlabel=None, ylabel=None):
    """
    General function to plot a boxplot between a categorical (x) and numerical (y) column.

    Parameters:
    - data (DataFrame): The dataset to visualize.
    - x_col (str): Name of the categorical column to be plotted on the x-axis.
    - y_col (str): Name of the numerical column to be plotted on the y-axis.
    - max_y (float, optional): If specified, filter out rows where y_col > max_y.
    - palette (str): Seaborn color palette to use.
    - title (str, optional): Custom plot title.
    - xlabel (str, optional): Label for the x-axis.
    - ylabel (str, optional): Label for the y-axis.
    """

    # Filter out values greater than max_y if provided
    if max_y is not None:
        data = data[data[y_col] <= max_y]

    # Set figure size
    plt.figure(figsize=(10, 6))

    # Create the boxplot
    sns.boxplot(data=data, x=x_col, y=y_col, palette=palette)

    # Set plot title and axis labels
    plt.title(title if title else f'Boxplot of {y_col} by {x_col}', fontsize=14)
    plt.xlabel(xlabel if xlabel else x_col, fontsize=12)
    plt.ylabel(ylabel if ylabel else y_col, fontsize=12)

    # Enable grid and layout
    plt.grid(True)
    plt.tight_layout()

    # Show the plot
    plt.show()

import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns

def create_count_plot(
    df, 
    x_column, 
    title="Count Plot", 
    x_label=None, 
    y_label="Count",
    figsize=(10, 6), 
    palette="viridis", 
    order=None,
    rotation=45, 
    annotate=False,
    horizontal=False,
    **kwargs
):
    """
    Creates a customizable count plot (bar plot of category frequencies).

    Parameters:
    -----------
    df : pandas.DataFrame
        Input DataFrame containing the data.
    x_column : str
        Column name for the categorical variable to count.
    title : str, optional
        Title of the plot (default: "Count Plot").
    x_label : str, optional
        Label for the x-axis (default: same as `x_column`).
    y_label : str, optional
        Label for the y-axis (default: "Count").
    figsize : tuple, optional
        Figure dimensions (width, height) in inches (default: (10, 6)).
    palette : str or list, optional
        Color palette (default: "viridis").
    order : list, optional
        Custom order of categories (default: None  auto-sorted).
    rotation : int, optional
        Rotation angle for x-axis labels (default: 45).
    annotate : bool, optional
        Whether to display counts on bars (default: False).
    horizontal : bool, optional
        If True, creates a horizontal bar plot (default: False).
    **kwargs : additional keyword arguments
        Passed to `sns.countplot()` for further customization.

    Returns:
    --------
    None (displays the plot).
    """
    plt.figure(figsize=figsize)
    
    if horizontal:
        ax = sns.countplot(
            y=x_column,
            data=df,
            palette=palette,
            order=order,
            **kwargs
        )
    else:
        ax = sns.countplot(
            x=x_column,
            data=df,
            palette=palette,
            order=order,
            **kwargs
        )
    
    # Customize labels
    plt.title(title, fontsize=15, pad=20)
    plt.xlabel(x_label if x_label else x_column, fontsize=12)
    plt.ylabel(y_label, fontsize=12)
    
    # Rotate x-axis labels if not horizontal
    if not horizontal:
        plt.xticks(rotation=rotation, ha='right' if rotation else 'center')
    
    # Add count annotations if enabled
    if annotate:
        for p in ax.patches:
            if horizontal:
                ax.annotate(
                    f"{int(p.get_width())}",
                    (p.get_width(), p.get_y() + p.get_height() / 2),
                    va="center",
                    xytext=(5, 0),
                    textcoords="offset points",
                )
            else:
                ax.annotate(
                    f"{int(p.get_height())}",
                    (p.get_x() + p.get_width() / 2, p.get_height()),
                    ha="center",
                    va="bottom",
                    xytext=(0, 5),
                    textcoords="offset points",
                )
    
    plt.tight_layout()
    plt.show()

Conclusions

There is no clear relationship between user age and trip duration, suggesting that age does not directly affect trip length.

Subscribers tend to make shorter, more rapid in-and-out trips, while customers make longer trips, particularly on weekends, which may reflect differences in usage patterns between the two groups.

Trip volume increases significantly on weekends, indicating increased use for leisure or non-daily activities.

Trip usage is primarily concentrated between 8 a.m. and 5-6 p.m., which corresponds to typical commuting times for work or school.
